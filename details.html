Overview

ShipBuilder is an open-source self-hosted PaaS. If you are familiar with Heroku, ShipBuilder offers the same features and functionality, with the added benefit of giving you full control over the hardware and environment where your applications run. ShipBuilder makes it easy to deploy (via git) and scale web-applications (via CLI) in the cloud or on physical hardware. This is a core component of SendHub's infrastructure; it runs all of the SendHub backend APIs as well as the frontend website, and allows application updates to be released seamlessly as often as desired.
